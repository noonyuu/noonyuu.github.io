개인정보취급방침

마지막 업데이트: 2026년 1월 8일

소개

모자피(이하 '본 앱')는 사용자의 개인정보를 존중하고 개인정보를 보호하기 위해 노력하고 있습니다. 본 개인 정보 보호 정책은 특히 얼굴 데이터 처리에 대해 자세히 설명합니다.

수집할 정보

이 앱은 다음 기능에 액세스합니다.

  • 카메라: 사진 촬영을 위한
  • 사진 라이브러리: 이미지 선택 및 저장을 위해

분석 및 광고 서비스의 정보 수집

본 앱은 서비스 개선 및 광고 게재를 위해 다음 타사 서비스를 사용합니다:

Firebase Analytics (Google)

앱 사용 현황을 분석하기 위해 Firebase Analytics를 사용합니다. 다음 정보가 자동으로 수집될 수 있습니다:

  • 기기 정보 (모델, OS 버전, 언어 설정 등)
  • 앱 사용 데이터 (실행 횟수, 사용 시간, 기능 사용 현황 등)
  • 대략적인 위치 정보 (국가/지역 수준)
  • 앱 충돌 정보

자세한 내용은 Google 개인정보처리방침을 참조하세요.

Google AdMob

광고 게재를 위해 Google AdMob을 사용합니다. AdMob은 다음 정보를 수집할 수 있습니다:

  • 기기 식별자 (광고 ID)
  • IP 주소
  • 기기 정보
  • 광고 표시 및 클릭 정보

맞춤 광고 설정은 기기 설정에서 변경할 수 있습니다. 자세한 내용은 Google 광고 정책을 참조하세요.

얼굴 데이터 처리 정보

중요: 이 앱은 얼굴 데이터를 수집, 저장 또는 외부로 전송하지 않습니다.

수집할 얼굴 데이터

본 앱은 얼굴 데이터를 수집하지 않습니다. Apple의 Vision 프레임워크를 사용하여 사용자의 장치에서 실시간으로 얼굴 위치(좌표 정보)를 감지하지만 이 정보는 임시 메모리에서만 사용되며 영구적으로 저장되지 않습니다.

얼굴 데이터 사용 목적

얼굴 위치 정보는 다음 용도로만 사용됩니다.

  • 이미지에서 얼굴의 위치를 ​​확인하고 모자이크 또는 이모티콘을 적용하려면
  • 사용자가 수동으로 선택한 얼굴 영역에 효과를 적용하기

이 모든 처리는 장치에서 로컬로 수행되며 처리가 완료되면 좌표 정보가 삭제됩니다.

제3자에 공유

얼굴 데이터는 제3자와 공유되지 않습니다. 본 어플리케이션은 네트워크 통신 기능을 가지지 않고, 얼굴의 위치 정보를 포함한 어떠한 데이터도 외부 서버에 송신하지 않습니다.

데이터 저장 위치

얼굴 위치 정보는 저장되지 않습니다. 처리하는 동안 장치의 메모리(RAM)에 일시적으로 존재하지만 앱을 닫거나 다른 이미지를 선택하면 즉시 삭제됩니다.

데이터 보유 기간

얼굴 위치 정보 보유 기간은 0입니다. 데이터는 영구적으로 저장되지 않으며 이미지 처리 세션 중에만 일시적으로 사용됩니다.

기술 구현 세부정보

본 앱은 다음 기술을 사용합니다.

  • Apple Vision Framework: 기기에서 얼굴 감지를 수행하는 Apple 정품 프레임워크
  • 기기 처리: 모든 이미지 처리가 사용자 기기에서 완료됩니다.
  • Firebase Analytics: Google에서 제공하는 앱 사용 현황 분석 서비스
  • Google AdMob: Google에서 제공하는 광고 게재 플랫폼

데이터 저장

이 앱은 이미지 데이터와 얼굴 데이터를 외부 서버에 저장하지 않습니다. 모든 처리는 장치에서 수행됩니다. 사용자가 저장을 선택한 경우에만 처리된 이미지가 장치의 사진 라이브러리에 저장됩니다.

제3자에게 제공

사용자의 개인 정보(얼굴 데이터 포함)는 제3자에게 판매, 대여 또는 공유하지 않습니다.

단, 위의 '분석 및 광고 서비스의 정보 수집' 섹션에 설명된 대로 앱 사용 현황 분석 및 광고 게재에 필요한 정보는 Firebase Analytics 및 Google AdMob을 통해 Google에 전송됩니다. 이러한 서비스의 데이터 처리에 대한 자세한 내용은 각 서비스의 개인정보처리방침을 참조하세요.
