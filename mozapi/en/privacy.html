Privacy Policy

Last updated: January 8, 2026

Introduction

Mozapi (hereinafter referred to as "this app") respects user privacy and strives to protect personal information. This Privacy Policy provides detailed information about how facial data is handled.

Information Collected

This app accesses the following features:

  • Camera: For taking photos
  • Photo Library: For selecting and saving images

Information Collection by Analytics and Advertising Services

This app uses the following third-party services for service improvement and ad delivery:

Firebase Analytics (Google)

We use Firebase Analytics to analyze app usage. The following information may be automatically collected:

  • Device information (model, OS version, language settings, etc.)
  • App usage data (number of launches, usage time, feature usage, etc.)
  • Approximate location information (country/region level)
  • App crash information

For more details, please refer to the Google Privacy Policy.

Google AdMob

We use Google AdMob for ad delivery. AdMob may collect the following information:

  • Device identifiers (Advertising ID)
  • IP address
  • Device information
  • Ad display and click information

You can change personalized ad settings from your device settings. For more details, please refer to Google Ads Policy.

About the Handling of Facial Data

Important: This app does not collect, store, or transmit facial data.

Facial Data Collected

This app does not collect facial data. It uses Apple's Vision framework to detect the location (coordinate information) of faces in real time on the user's device. However, this information is used only in temporary memory and is not permanently stored.

Purpose of Use of Facial Data

Facial location information is used only for the following purposes:

  • To identify the location of faces in an image and apply a mosaic or emoji
  • To apply effects to facial areas manually selected by the user

All of this processing is performed locally on the device, and the coordinate information is discarded once processing is complete.

Sharing with Third Parties

Facial data is not shared with third parties. This app does not have network communication capabilities and does not send any data, including facial location information, to external servers.

Data Storage Location

Face location information is not stored. It temporarily resides in your device's memory (RAM) during processing, but is immediately discarded when you close the app or select another image.

Data Retention Period

Face location information has a retention period of zero. The data is not stored permanently; it is used temporarily only during the image processing session.

Technical Implementation Details

This app uses the following technologies:

  • Apple Vision Framework: Apple's native framework for on-device face detection
  • On-Device Processing: All image processing is completed on the user's device
  • Firebase Analytics: Analytics service provided by Google for app usage analysis
  • Google AdMob: Advertising platform provided by Google for ad delivery

Data Storage

This app does not store image or facial data on external servers. All processing is performed on the device. Processed images are saved to the device's photo library only if the user chooses to save them.

Provision to Third Parties

We will not sell, rent, or share your personal information (including facial data) with third parties.

However, as described in the "Information Collection by Analytics and Advertising Services" section above, information necessary for app usage analysis and ad delivery is sent to Google through Firebase Analytics and Google AdMob. For details on how these services handle data, please refer to their respective privacy policies.
